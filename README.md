# 💰 Finance Tracker App
A full-stack fintech-style application that allows users to track income and expenses, visualize spending, and manage personal finances securely.

## 🚀 Features
- 🔐 Secure Authentication (JWT + HTTP-only cookies)
- 📊 Dashboard with analytics (Income vs Expenses)
- 📈 Charts (Bar + Doughnut)
- ➕ Add transactions with categories
- 👤 User profile page
- 🚪 Logout functionality
- 🎨 Clean modern UI (Tailwind CSS)

## 🛠️ Tech Stack
- Frontend: Next.js (App Router), React, Tailwind CSS
- Backend: Next.js API Routes
- Database: MongoDB (Mongoose)
- Authentication: JWT (stored in HTTP-only cookies)
- Charts: Chart.js
